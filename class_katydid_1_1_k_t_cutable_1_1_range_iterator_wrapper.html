Detailed Description

template<class XObjectType, class XRangeIteratorType>
class Katydid::KTCutable< XObjectType, XRangeIteratorType >::RangeIteratorWrapper

Combines the RangeIterator with its original position and a < operator.

Author
N. S. Oblath

The RangeIterator is the original type of iterator given to KTCutable. This class wraps that iterator with other information and capabilities:

  • fPosition is the iterators position in the original range given to KTCutable.
  • operator< allows a RangeIteratorWrapper to be compared to other RangeIteratorWrappers based on position.
