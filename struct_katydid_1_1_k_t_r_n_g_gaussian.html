Detailed Description

template<typename Engine = KTGlobalRNGEngine, typename RealType = double>
class Katydid::KTRNGGaussian< Engine, RealType >

Gaussian distribution with parameters "mean" and "sigma".

Author
N. S. Oblath

Returns a floating point value distributed according to a Gaussian distribution with parameters "mean" and "sigma."

The PDF for the distribution is:

Available configuration options:

  • "mean": double – Mean of the Gaussian distribution
  • "sigma": double – Standard deviation of the Gaussian distribution
