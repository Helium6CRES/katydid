Detailed Description

Implementation of a Hann window function.

Author
N. S. Oblath

Name for specifying this window function: hann

The Hann window is a general-purpose windowing function often used to select a subset of samples for performing a Fourier Transform. It has the advantage of low aliasing with a fairly small loss of resolution. Source: Wikipedia: Hann function (http://en.wikipedia.org/wiki/Hann_function, 8/1/2012)

Available configuration values: none
