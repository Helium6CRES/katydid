Detailed Description

Implementation of a Hamming window function.

Author
N. S. Oblath

Name for specifying this window function: hamming

The Hamming window has the same functional form as the Hann window, with parameters optimized to minimize the nearest side lobe (side lobe height is about 1/5 that of the Hann window). Source: Wikipedia: Window function (http://en.wikipedia.org/wiki/Window_function#Hamming_window, 8/1/2012)

Available configuration values: none
