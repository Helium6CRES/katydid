Definition at line 110 of file KTWignerVille.cc.

KTSlidingWindowFSDataFFTW* TransformData ( const KTTimeSeriesData *  data)

Performs the W-V transform on the given time series data. In the process, the data is FFTed, and then reverse FFTed; if you want to keep the intermediate frequency spectrum, pass a KTFrequencySpectrumDataFFTW** as the second parameter..

Note
A frequency spectrum data object can still be returned even if the full W-V transform fails!
