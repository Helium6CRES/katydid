Detailed Description

Iterates over the bundles in an Egg file.

Author
N. S. Oblath

Iterates over bundles in an egg file; bundles are extracted until fNBundles is reached.

Available configuration options:

  • "number-of-bundles": UInt_t – Number of bundles to process
  • "filename": string – Egg filename to use
  • "egg-reader": string – Egg reader to use (options: monarch [default], 2011)
  • "time-series-size": UInt_t – Specify the size of the time series (select 0 to use the Monarch record length)
  • "time-series-type": string – Type of time series to produce (options: real [default], fftw [not available with the 2011 egg reader])
  • "output-data-name": string – Name to give to the data produced

Command-line options defined

  • -n (n-bundles): Number of bundles to process
  • -e (egg-file): Egg filename to use
  • -z (–use-2011-egg-reader): Use the 2011 egg reader

Signals:

  • "header": void (const KTEggHeader*) – emitted when the file header is parsed.
  • "data": void (const KTTimeSeriesData*) – emitted when the new time series is produced.
  • "bundle": void (boost::shared_ptr<KTBundle>) – emitted when an bundle is read from the file.
  • "egg-done": void () – emitted when a file is finished.
