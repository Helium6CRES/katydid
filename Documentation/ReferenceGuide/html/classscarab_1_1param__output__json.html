Detailed Description

Convert Param to JSON.

Author
N.S. Oblath

Options:

  • JSON Style Pretty print: { "style": param_output_json::k_pretty } or { "style": "pretty" } Compact (default): { "style": param_output_json::k_compact } or { "style": "compact" } or anything else
