Detailed Description

Custom event loop for ROOT GUIs.

Author
N. S. Oblath

Development note concerning ROOT issues: The CINT dictionary generator can't know that this class inherits from a non-ROOT class (KTEventLoop), but a dictionary must be generated for use of the signal/slot mechanism. Therefore references to KTEventLoop are contained in #ifndef blocks.
