Detailed Description

Mantis client.

Author
N. Oblath

Configuration name: "mantis-client"

Available configuration values:

  • "slice-size": unsigned – Specify the size of the time series (required)
  • "stride": unsigned – Specify how many bins to advance between slices; must be equal to or larger than the slice size Leave unset to make stride == slice-size; i.e. no overlap or skipping between slices
  • "client": object – Mantis client configuration values (see Mantis documentation for more information)

Slots:

Signals:

  • "header": void (KTEggHeader*) – emitted when the file header is parsed.
  • "raw-ts" void (Nymph::KTDataPtr) – emitted when a new raw time series is produced; guarantees KTRawTimeSeriesData
  • "mantis-done": void () – emitted when a file is finished.
