Detailed Description

Implementation of a rectangular window function.

Author
N. S. Oblath

Within the window area, all weights are 1. Outside of the window area, the weights are 0.

Available configuration values:

  • boxcar-size: unsigned – size (# of bins) of the interval where the weight is 1; this interval is centered in the window
