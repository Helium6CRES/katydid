Detailed Description

template<class XObjectType, class XRangeIteratorType>
class Katydid::KTCutable< XObjectType, XRangeIteratorType >::iterator

A bi-directional iterator over the interval that skips any cut objects.

This iterator will traverse the specified range of objects, skipping any that have been cut out.

This is a bi-directional iterator, but it will adopt some of the characteristics of the original iterator passed to KTCutable. For instance, if XRangeIteratorType is a reverse iterator, then KTCutable::iterator will act in reverse.
