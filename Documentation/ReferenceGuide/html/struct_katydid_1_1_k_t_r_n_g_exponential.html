Detailed Description

template<typename Engine = KTGlobalRNGEngine, typename RealType = double>
class Katydid::KTRNGExponential< Engine, RealType >

Exponential distribution with parameter "lambda".

Author
N. S. Oblath

Returns a floating point value distributed according to an exponential distribution with parameter \(\lambda \equiv\) "lambda."

The PDF for the distribution is: \( p(x) = \lambda\exp{-\lambda{}x} \)

Available configuration options:

  • "lambda": double – rate parameter of the exponential distribution
