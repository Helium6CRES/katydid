Detailed Description

template<size_t NDims>
class Katydid::KTAxisProperties< NDims >

Encapsulates the properties of n-dimensional binned axes.

Author
N. S. Oblath

Provides the number of bins and axis ranges for n-dimensional axes. This is intended to be combined with array- or vector-like storage classes.

Note
Dimensions are numbered on the interval [1, NDims].
Template Parameters
<NDims>{Number of dimensions}
