Detailed Description

template<typename Engine = KTGlobalRNGEngine, typename IntType = int, typename RealType = double>
class Katydid::KTRNGPoisson< Engine, IntType, RealType >

Poisson distribution with parameter "mean".

Author
N. S. Oblath

Returns an integer value distributed according to a Poisson distribution with parameter \(\lambda \equiv\) "mean."

The PDF for the distribution is: \( p(i) = \frac{exp{-\lambda}\lambda^i}{i!} \)

Available configuration options:

  • "mean": double – Mean of the Poisson distribution
