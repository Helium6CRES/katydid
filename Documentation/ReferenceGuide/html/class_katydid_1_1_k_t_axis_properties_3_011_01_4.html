Detailed Description

template<>
class Katydid::KTAxisProperties< 1 >

Specialization of KTAxisProperties; encapsulates the properties of 1-dimensional binned axes.

Author
N. S. Oblath

Provides the number of bins and axis ranges for 1-dimensional axis. This is intended to be combined with an array- or vector-like storage class.
