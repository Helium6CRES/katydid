Detailed Description

Performs a chi-squared test on the spline fit to an accumulated power spectrum.

Author
E. Zayas

Performs a chi-squared test on the spline fit to an accumulated power spectrum. Returns chi-squared value and NDF

Available configuration values:

  • MinFrequency: minimum frequency bound for chi-squared calculation
  • MaxFrequency: maximum frequency bound for chi-squared calculation
  • MinBin: bin associated with minimum frequency bound
  • MaxBin: bin associated with maximum frequency bound

Slots:

  • "ps": void (Nymph::KTDataPtr) – Requires KTPowerSpectrumData, KTPowerSpectrumUncertaintyData and KTGainVariationData; Adds KTGainVarChi2Data

Signals:

  • "chi2-test": void (Nymph::KTDataPtr) Emitted upon chi-squared and NDF calculation; Guarantees KTGainVarChi2Data
