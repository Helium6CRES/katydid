Detailed Description

Performs an FFT-based electron hunt.

Author
N. S. Oblath

Uses a windows FFT of Egg bundles to search for clusters of high-peaked bins moving up in frequency.

Available configuration values:

  • output-filename_base –
  • threshold-multiplier –
  • write-text-file –
  • draw-waterfall –
  • group-bins-margin-high –
  • group-bins-margin-low –
  • group-bins-margin-same_time –
  • input-data-name – name used to find data when processing an bundle
